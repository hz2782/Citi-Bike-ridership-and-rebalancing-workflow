2.data tidying

Next, clean the variables holding percentages as a type char. I’ll do this by removing the % and recasting the variables, all in one set of piping functions:

d <- d %>% 
  mutate( date = as.Date(date, format = '%m/%d/%Y') ) %>%
  mutate( mta_total = as.numeric(mta_total) ) %>%
  mutate_if( is.character, str_replace_all, pattern = '%', replacement = '' ) %>%
  mutate_if( is.character, as.numeric )

5.data interpretation

The NYT article did not include changes in ridership for the Citi Bike bike share. Since I graphed the changes in Citi Bike bike rides alongside the transportation types we just graphed, it showed that Citi bike shares similar flactuations as the transportation types in our version. This is because although other transportation types have different values of percentage decline, they follow the same trend over time. Therefore, as a type of transportation, I think the Citi Bike follows the same pattern.
